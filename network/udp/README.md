udp - network
===============

C network sample <br/>
send and recieve UDP packet  using socket functions <br/>

### sample code
- udp_send.c <br/>
simple UDP transmitter <br/>
- udp_recv.c <br/>
simple UDP receiver <br/>

build code  <br/>
% gcc udp_send.c <br/>

run code <br/>
% /a.out   <br/>

display as below <br/>
> sendto: 127.0.0.1 : 12345  <br/>
