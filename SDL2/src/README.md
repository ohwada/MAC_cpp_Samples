SDL2 src
===============

SDL2 Sample <br/>

## samples
- window.c  <br/>
draw simple window  <br/>

- font.c  <br/>
draw text with font  <br/>

- image.c  <br/>
draw image  <br/>

- animate.c  <br/>
animate image  <br/>

- key_event.c  <br/>
 move the image with arrow key  <br/>

- mouse_event.c  <br/>
 move the image with mouse click  <br/>
 
Build and Run  <br/>
% gcc window.c -o window  `pkg-config --cflags --libs sdl2`   <br/>
