### Installing:

- SDL <br/>
- SDL_image <br/>

% make mac <br/>

### How to play
This is not strictly Game <br/>
You taste the atmosphere of Super Mario Bros <br/>
You can move Mario sideways with Arrow Key.<br/>
You cannot jump <br/>
