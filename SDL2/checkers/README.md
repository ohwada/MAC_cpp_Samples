### About: 
This is a Checkers Game built with SDL2 in C++.<br/>

https://en.wikipedia.org/wiki/Checkers <br/>

### Installing:
Require : <br/>
- SDL2 <br/>
- SDL2_ttf <br/>
- OpenGL <br/>
- glm <br/>

% make <br/>

### How to Play
Two Players Game .<br/>
No match mode with the Computer .<br/>
Game rules are taken from the international variant of draughts.  .<br/>
