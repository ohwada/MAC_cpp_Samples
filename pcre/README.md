pcre
===============

pcre:  
Perl compatible regular expressions library  
https://www.pcre.org/  

### Install pcre
% brew pcre  
https://formulae.brew.sh/formula/pcre  

% pkg-config --cflags --libs libpcre  

### Install pcre2

% brew pcre2  
https://formulae.brew.sh/formula/pcre2  

% pkg-config --cflags --libs libpcre2-8
