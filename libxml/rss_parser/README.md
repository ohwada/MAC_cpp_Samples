libxml: rss parser
===============

RSS Parser <br/>
extract the Title and pubDate from RSS Feeds  <br/>

Sample Feeds <br/>
https://www.feedforall.com/sample-feeds.htm <br/>

Build and Run <br/>
% make <br/>


% ./rss sample.xml <br/>
