example_image - SDL
===============

SDL Sample <br/>
draw png image and bmp image from image file <br/>


## build sample code
% g++ *.cc -o mysdl `pkg-config --cflags --libs sdl2_image` <br/>

## run sample code
% ./mysdl <br/>

draw 'P', when enter key 'p' <br/>
draw 'B', when enter key 'b' <br/>
