vmail
===============

vmail Sample <br/>
send and recieve mail <br/>

vmail : <br/>
a powerful C++ class library for working with RFC-822 and MIME messages and Internet messaging services like IMAP, POP or SMTP. <br/>
https://www.vmime.org/ <br/>
