### Install   
(1) Downloas souce file  
% git clone https://github.com/mattn/tinytinyhttpd  

(2) create tthttpd
cd downloaded directory(tinytinyhttpd)
% ./autogen.sh
% ./configure
% make
> created tthttpd

(3) copy tthttpd to Your directpry

(4) run tthttpd
% ./tthttpd -h
> tthttpd (tinytinyhttpd) 0.0.11
> usage: tthttpd
