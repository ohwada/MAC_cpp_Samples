Class Inheritance
===============

C++ Sample of Class Inheritance <br/>

- animal.cpp <br/>
Animal class: base class <br/>

- dog.cpp  <br/>
Dog class: inheritance class <br/>

- Cat.cpp  <br/>
Cat class: inheritance class <br/>

Build and Run <br/> 
%  g++ *.cpp -std=c++11 -o animal <br/> 
% ./animal <br/> 

display as below  <br/> 
> Animal: milk: 1: moo <br/> 
> Dog: taro: 2: waoon <br/> 
> Cat: tama: 3: nya <br/>
