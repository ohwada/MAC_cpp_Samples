libxml++: text reader
===============

TextReader Parser <br/>
This example examines each node in turn, then moves to the next node. <br/>

Build and Run <br/>
% g++  main.cc -std=c++11 `pkg-config --cflags --libs libxml++-2.6` -o text <br/>

% ./text <br/>
> --- node --- <br/>
> name: example <br/>
> depth: 0 <br/>
> no attributes <br/>
> no value <br/>
