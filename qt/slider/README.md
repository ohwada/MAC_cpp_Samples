QT Sample <br/>

<image src="https://raw.githubusercontent.com/ohwada/MAC_cpp_Samples/master/qt/slider/slider.png"  /> <br/>

Build and Run <br/>
(1) % qmake -project <br/>

(2) add the following to the end of slider.pro <br/>

> QT+=widgets <br/>

(3) % qmake <br/>

(4) % make <br/>

(5)% ./slider.app/Contents/MacOS/slider <br/>
