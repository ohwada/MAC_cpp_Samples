<image src="https://raw.githubusercontent.com/ohwada/MAC_cpp_Samples/master/qt/finddialog/finddialog.png"  /> <br/>

Build and Run <br/>
(1) % qmake -project <br/>

(2) add the following to the end of  finddialog.pro <br/>

> QT+=widgets <br/>

(3) % qmake <br/>

(4) % make <br/>

(5)% ./finddialog.app/Contents/MacOS/finddialog <br/>
