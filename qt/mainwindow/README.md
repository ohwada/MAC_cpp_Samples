Build and Run <br/>
(1) % qmake -project <br/>

(2) add the following to the end of mainwindow.pro <br/>

> QT+=widgets <br/>

(3) % qmake <br/>

(4) % make <br/>

(5) % ./maainwindow.app/Contents/MacOS/mainwindow <br/>
