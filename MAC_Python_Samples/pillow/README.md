pillow
===============

[pypi: pillow](https://pypi.org/project/pillow/)

Python Imaging Library (Fork)

 ## Installation

> % pip3 install pillow

## Features

- Pillow has No Image Viewer    

using Image Viewer enbded in platform

Preview App For Mac

- suport jpeg  

 - resize image
