tkinter
===============

[homebrew: python-tk@3.12](https://formulae.brew.sh/formula/python-tk@3.12)

Python interface to Tcl/Tk

## Installation

> % brew install python-tk@3.12

## Features

- load Image file,  Not suport jpeg  

 - No Feature to resize Image

use Pillow
