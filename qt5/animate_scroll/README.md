QT Animate Scroll 
===============

QT Sample <br/>
 animate scroll window <br/>

Build and Run <br/>
(1) % qmake -project <br/>

(2) add the following to the end of  animate_scroll.pro <br/>

> QT+=widgets <br/>

(3) % qmake <br/>

(4) % make <br/>

(5) double click  animate_scrollarea.app <br/>
