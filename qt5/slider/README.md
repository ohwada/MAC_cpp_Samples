QT Slider
===============

QT Sample <br/>
draw slider and button <br/>

<image src="https://raw.githubusercontent.com/ohwada/MAC_cpp_Samples/master/qt5/slider/slider.png" > <br/>

Build and Run <br/>
(1) % qmake -project <br/>

(2) add the following to the end of slider.pro <br/>

> QT+=widgets <br/>

(3) % qmake <br/>

(4) % make <br/>

(5) double click slider.app <br/>
