Build and Run <br/>
(1) % qmake -project <br/>

(2) add the following to the end of ScrollArea.pro <br/>

> QT+=widgets <br/>

(3) % qmake <br/>

(4) % make <br/>

(5) double click ScrollArea.app <br/>
