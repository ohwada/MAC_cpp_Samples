gtk3: control
===============

<image src="https://raw.githubusercontent.com/ohwada/MAC_cpp_Samples/master/gtk3/screenshots/check_button.png" width="300" /> 

Widgets that can be manipulated by the user are called Controls.

### samples
- button.c
- checkbutton.c
- comboboxtext.c
- listbox.c
- radiobutton.c
- scale.c
- togglebutton.c

Build and Run:  
% make button
% bin/button
