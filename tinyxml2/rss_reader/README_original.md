RSS Reader
==============

Simple RSS Reader in C++ using TinyXML library.

##Required:##

```apt-get install libtinyxml2.6.2```

##Usage:##

```./rss_reader XML_FILENAME ...```



##Under development:##

- Better compatibility with RSS2
- GUI with Qt4
