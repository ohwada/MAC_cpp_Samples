SDL_mixer
===============

SDL mixer Sample <br/>
play music file <br/>

## install SDL_mixer

see <br/>
https://github.com/ohwada/MAC_cpp_Samples/tree/master/SDL_mixer/setup.md  <br/>

## play_music_file.c
play music file <br/>

build file <br/>
% gcc play_music_file.c -o play  `pkg-config --cflags --libs sdl_mixer` <br/>

run <br/>
%  ./play media/m-art_Shine.ogg
